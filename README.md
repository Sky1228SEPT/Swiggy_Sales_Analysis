# Swiggy_Sales_Analysis
Analyzing Swiggy sales data using Python with data preprocessing, EDA, and visualization. Built dashboards using Matplotlib &amp; Seaborn to identify sales trends, top-performing categories, customer patterns, and actionable business insights.

## 📌 Overview

This project focuses on performing an end-to-end **Sales Data Analysis** of Swiggy food delivery data using Python.  
The objective of this project is to extract meaningful business insights by analyzing customer orders, sales trends, restaurant performance, and revenue patterns.

The project includes:
- Loading and understanding the dataset
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization and dashboard creation
- Generating analytical reports
- Preparing a business presentation using Gamma

This project demonstrates practical Data Analyst skills using Python libraries and visualization techniques.
